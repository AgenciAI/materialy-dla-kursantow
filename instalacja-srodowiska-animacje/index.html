Jak zainstalować — 3 kroki

Skrypt zainstaluje wszystko automatycznie

1
Rozpakuj folder
Wypakuj pobrany ZIP w dowolnym miejscu na dysku — np. na Pulpicie.
2
Uruchom skrypt instalacyjny
Otwórz Terminal, przejdź do folderu i uruchom:

cd ~/Desktop/animacje-claude-code
bash install.sh

Skrypt zainstaluje Homebrew, Node.js, FFmpeg, Claude Code CLI i wszystkie skille. Przy Homebrew może zapytać o hasło administratora — to normalne.
3
Gotowe — uruchom Claude Code
Po instalacji otwórz dowolny folder w Terminalu i wpisz:

claude

Opisz co chcesz zrobić — Claude dobierze narzędzie, zaproponuje koncepcje i poprowadzi Cię przez cały proces.
⚠ Instalacja video-use pobiera ~314MB danych — upewnij się że masz połączenie z internetem. Cały proces może trwać 5–10 minut.

Jak Claude pracuje z animacjami
W
Obowiązkowy workflow przy każdej animacji
1
Rekomendacja narzędzia
Claude pyta co chcesz osiągnąć i rekomenduje HyperFrames lub Remotion z uzasadnieniem. Ty decydujesz czy się zgadzasz.
2
Warstwa audio przez ElevenLabs
Claude pyta czy chcesz dodać: głos lektora / efekty dźwiękowe / muzykę w tle / kombinację / bez dźwięku. Uwzględnia to już w koncepcjach.
3
3 koncepcje wizualne przed kodem
Claude proponuje 3 różne koncepcje (nazwa, opis wizualny, nastrój, technika, czas). Nie pisze kodu bez Twojego wyboru.
4
Podgląd w przeglądarce
Po napisaniu kodu Claude uruchamia podgląd i czeka na Twoje potwierdzenie że wygląda dobrze.
5
Render do MP4
Dopiero po potwierdzeniu — Claude renderuje do pliku MP4.
ElevenLabs — głos, muzyka, efekty dźwiękowe
E
Konfiguracja ElevenLabs API
wymagane dla audio w animacjach
  1. Zarejestruj konto na elevenlabs.io — darmowy tier wystarczy na start
  2. Wejdź w Settings → API Keys i skopiuj klucz
  3. Claude zapyta o klucz przy pierwszym użyciu audio — wklej go wtedy

ElevenLabs umożliwia generowanie głosu lektora (voiceover), efektów dźwiękowych (SFX) i muzyki w tle zsynchronizowanej z animacją.

Remotion — nowy projekt wideo
i
Remotion instaluje się per projekt

Remotion to framework React — każde wideo to osobny projekt Node.js. Poniższe komendy wykonujesz gdy Claude zasugeruje Remotion i zdecydujesz się go użyć.

R1
Utwórz nowy projekt
npx create-video@latest
szablon Wybierz Blank
TailwindCSS Wybierz Yes
Skills Wybierz Yes
R2
Zainstaluj zależności i uruchom Studio
cd my-video npm install npm run dev

Studio otwiera się pod localhost:3000 — podgląd animacji klatkę po klatce.

R3
Render do MP4
po zatwierdzeniu podglądu
npx remotion render src/index.ts NazwaKompozycji out/video.mp4

⚠ Renderuj dopiero po sprawdzeniu animacji w Studio i potwierdzeniu że wygląda dobrze.
