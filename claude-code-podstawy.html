Podstawy Claude Code — Dla Początkujących

Claude Code to narzędzie do pracy z AI bezpośrednio w terminalu Twojego komputera. Nie potrzebujesz żadnego interfejsu graficznego — piszesz komendy, a Claude rozumie kontekst Twojego projektu, czyta pliki, edytuje kod i wykonuje zadania. Ten przewodnik poprowadzi Cię od zera: od instalacji wymaganych programów, przez pierwsze uruchomienie, aż po poznanie wszystkich dostępnych komend. Jeśli nigdy wcześniej nie korzystałeś z terminala — nie martw się. Wszystko jest tu wyjaśnione krok po kroku.

Krok 0 / Podstawy

Co to jest terminal?

Terminal to okno, w którym komunikujesz się z komputerem przez tekst. Zamiast klikać myszą w ikony i menu — piszesz komendy, a komputer je wykonuje. Claude Code działa właśnie w terminalu, dlatego zacznijmy od tego jak go otworzyć.

🍎

Otwórz terminal — macOS

Najszybszy sposób: naciśnij ⌘ Cmd + Spacja, wpisz Terminal i naciśnij Enter.

Alternatywnie: Finder → Programy → Narzędzia → Terminal.

Otworzy się okno z kursorem i znakiem $ — to jest terminal. Czeka na Twoją komendę.

🪟

Otwórz terminal — Windows

Naciśnij Win + R, wpisz cmd i naciśnij Enter — otworzy się klasyczny wiersz poleceń.

Lub: naciśnij Win, wpisz PowerShell i otwórz Windows PowerShell.

Zalecane: zainstaluj Windows Terminal ze sklepu Microsoft Store — nowocześniejszy interfejs.

Jak działa terminal? Kiedy widzisz znak $ lub > na końcu linii — terminal czeka na Twoją komendę. Piszesz komendę, naciskasz Enter, terminal ją wykonuje i wyświetla wynik. Każda komenda w tym przewodniku jest gotowa do skopiowania i wklejenia. Możesz wklejać używając ⌘ Cmd + V (macOS) lub Ctrl + V (Windows).
Krok 0 / Zrozumienie

Jak działa Claude Code?

Claude Code to program instalowany na komputerze i uruchamiany w terminalu. Po uruchomieniu otwiera się sesja konwersacji z AI — podobna do czatu, ale znacznie potężniejsza. Kluczowa różnica względem Claude.ai w przeglądarce:

Możliwość Claude.ai
(przeglądarka)		 Claude Code
(terminal)
Dostęp do Twoich plików✗✓
Czyta cały folder projektu✗✓
Edytuje pliki na dysku✗✓
Uruchamia komendy i skrypty✗✓
Pamięta kontekst projektu✗✓
Interfejs graficzny✓✗ (terminal)
Ważna zasada: Claude Code „widzi" folder, w którym go uruchomisz. Dlatego zanim zaczniesz — zawsze przechodzisz do folderu swojego projektu komendą cd, a dopiero potem uruchamiasz claude. Claude pyta o zgodę przed każdą modyfikacją pliku.
Krok 1 / Wymagania

Zainstaluj to najpierw

Claude Code wymaga Node.js w wersji 18 lub nowszej. Na macOS zalecamy zacząć od Homebrew — menedżera pakietów, który ułatwia instalację wszystkich pozostałych narzędzi.

🍺
tylko macOS

Homebrew

Homebrew to menedżer pakietów dla macOS — działa jak „sklep z aplikacjami dla terminala". Jedną komendą instalujesz dowolny program. Zainstaluj go jako pierwsze.

terminal — macOS 
/bin/bash -c "$(curl -fsSL \
  https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Instalator może zapytać o hasło do komputera — wpisz je (znaki nie będą widoczne, to normalne) i naciśnij Enter. Instalacja trwa 2–5 minut.

weryfikacja 
brew --version
🟢
wymagane

Node.js

Node.js to silnik, który pozwala uruchomić Claude Code. Nie musisz znać JavaScript — to tylko środowisko uruchomieniowe. Wymagana wersja: 18 lub nowsza.

macOS (przez Homebrew):

terminal — macOS 
brew install node

Windows (przez winget):

terminal — Windows 
winget install OpenJS.NodeJS.LTS

Ręcznie (macOS i Windows): pobierz wersję LTS z nodejs.org i uruchom instalator.

weryfikacja 
node --version
npm --version
🔀
opcjonalne

Git

Git to system kontroli wersji — śledzi zmiany w plikach i pozwala wrócić do poprzednich wersji kodu. Claude Code działa bez Gita, ale jeśli pracujesz z projektami programistycznymi lub chcesz żeby Claude commitował zmiany — Git będzie potrzebny.

sprawdź czy masz git 
git --version

macOS:

terminal — macOS 
brew install git

Windows: pobierz instalator z git-scm.com i przejdź przez kreatora.

Krok 2 / Instalacja

Instalacja Claude Code

Claude Code instalujesz przez npm — menedżer pakietów Node.js. To jedna komenda. Działa identycznie na macOS i Windows.

1

Instalacja globalna

Otwórz terminal i wklej tę komendę. Flaga -g instaluje Claude Code globalnie — będzie dostępny z każdego folderu.

terminal 
npm install -g @anthropic-ai/claude-code
macOS — błąd permission denied? Jeśli zobaczysz błąd EACCES lub permission denied, użyj:

sudo npm install -g @anthropic-ai/claude-code

Terminal zapyta o hasło do komputera — wpisz je i naciśnij Enter.
2

Weryfikacja instalacji

Sprawdź czy Claude Code jest poprawnie zainstalowany:

terminal 
claude --version

Jeśli wyświetli się numer wersji — instalacja się powiodła. Możesz przejść dalej.

3

Przejdź do folderu projektu

Przed uruchomieniem Claude Code zawsze przejdź do folderu, na którym chcesz pracować. Komenda cd zmienia bieżący folder w terminalu.

macOS — przykład 
cd ~/Desktop/moj-projekt
Windows — przykład 
cd C:\Users\TwojaNazwa\Desktop\moj-projekt
Nie masz jeszcze projektu? Możesz uruchomić Claude Code w dowolnym pustym folderze lub nawet na pulpicie — po prostu cd ~/Desktop. Claude zobaczy co jest w tym folderze.
4

Pierwsze uruchomienie i logowanie

Uruchom Claude Code wpisując:

terminal 
claude

Przy pierwszym uruchomieniu Claude Code poprosi Cię o logowanie. Możesz zalogować się przez:

  • Konto Anthropic (claude.ai) — kliknij link, który pojawi się w terminalu, i zatwierdź w przeglądarce
  • Klucz API — jeśli masz klucz API Anthropic, wybierz tę opcję i wklej klucz

Po zalogowaniu zobaczysz znak zachęty. Możesz zacząć pisać zadania w języku naturalnym.

5

Pierwsze zadanie

Po wejściu do sesji Claude Code wpisz swoje pierwsze polecenie po polsku:

przykładowe pierwsze polecenie 
Opisz mi strukturę tego projektu.

Claude przeanalizuje pliki w folderze i wyjaśni co widzi. Możesz pisać po polsku — Claude rozumie oba języki.

Żeby wyjść z sesji: wpisz /exit lub naciśnij Ctrl + C dwukrotnie.
Krok 3 / Claude Desktop

Claude Desktop — aplikacja na komputer

Claude Desktop to osobna aplikacja z interfejsem graficznym — odpowiednik claude.ai, ale zainstalowany lokalnie. Działa niezależnie od Claude Code. Możesz używać obu jednocześnie. Największa zaleta: obsługuje MCP (Model Context Protocol) — protokół pozwalający podłączyć Claude do zewnętrznych narzędzi i baz danych.

💻

Jak zainstalować

  1. Wejdź na claude.ai/download
  2. Wybierz wersję dla swojego systemu (macOS lub Windows)
  3. Pobierz plik i uruchom instalator:
    macOS: przeciągnij ikonę do folderu Aplikacje
    Windows: przejdź przez kreatora instalacji
  4. Otwórz aplikację i zaloguj się kontem Anthropic
⚡

Co to MCP?

MCP (Model Context Protocol) to protokół, który pozwala podłączyć Claude Desktop do zewnętrznych narzędzi. Dzięki MCP możesz dać Claude dostęp do swoich plików, Notion, GitHub, Airtable, własnej bazy danych i wielu innych systemów.

Claude Code i Claude Desktop to dwa różne narzędzia — Claude Code to terminal dla zaawansowanych użytkowników, Claude Desktop to aplikacja dla każdego.

Krok 4 / Komendy

Wszystkie komendy

Komendy Claude Code dzielą się na dwie grupy: slash commands — używane wewnątrz sesji (po uruchomieniu claude), oraz flagi CLI — używane przy uruchamianiu Claude Code w terminalu.

Slash commands — komendy wewnątrz sesji

Po uruchomieniu claude i wejściu do sesji możesz używać komend zaczynających się od /. To polecenia dla samego Claude Code, nie dla AI.

KomendaCo robi
/helpWyświetla listę wszystkich dostępnych komend
/clearCzyści historię konwersacji w bieżącej sesji
/configOtwiera ustawienia Claude Code
/permissionsZarządza uprawnieniami — co Claude może robić bez pytania
/agentsWyświetla dostępnych agentów i pozwala nimi zarządzać
/hooksZarządza hookami — automatyczne akcje przy zdarzeniach
/doctorSprawdza czy instalacja jest poprawna i nic nie brakuje
/modelZmienia model AI w trakcie sesji (Sonnet, Opus, Haiku)
/costWyświetla szacunkowy koszt zużytych tokenów w sesji
/compactKompresuje historię rozmowy — oszczędza tokeny przy długich sesjach
/reviewUruchamia przegląd kodu — Claude ocenia jakość zmian
/initGeneruje plik CLAUDE.md w projekcie — stały kontekst dla AI
/memoryZarządza trwałą pamięcią Claude Code między sesjami
/bugOtwiera formularz zgłoszenia błędu do Anthropic
/loginLoguje do konta Anthropic lub zmienia aktywne konto
/logoutWylogowuje z aktywnego konta Anthropic
/quit lub /exitKończy sesję Claude Code i wraca do terminala

Flagi CLI — opcje przy uruchamianiu

To opcje, które dodajesz do komendy claude w terminalu — przed wejściem do sesji.

KomendaCo robi

claude --dangerously-skip-permissionsPomija wszystkie prośby o potwierdzenie — używaj ostrożnie

Krok 5 / Kontekst projektu

CLAUDE.md — stały kontekst projektu

Kiedy uruchamiasz Claude Code w folderze projektu, automatycznie wczytuje on plik CLAUDE.md. To plik tekstowy, w którym opisujesz swój projekt — co to jest, jak działa, jakich zasad ma się trzymać AI. Dzięki temu nie musisz za każdym razem tłumaczyć od nowa kontekstu.

⚡

Wygeneruj automatycznie

Uruchom Claude Code w folderze projektu i wpisz komendę:

wewnątrz sesji 
/init

Claude przeanalizuje projekt i wygeneruje plik CLAUDE.md z podstawowym opisem struktury, komend i zasad.

✏️

Utwórz ręcznie

Możesz też stworzyć plik samodzielnie — utwórz CLAUDE.md w głównym folderze projektu i opisz projekt prostym językiem.

Przykładowa zawartość:

CLAUDE.md 
# Mój projekt

To jest aplikacja do zarządzania zadaniami.
Używamy Next.js i Tailwind CSS.
Baza danych: PostgreSQL.

Zasady: nie usuwaj istniejących plików
bez pytania. Używaj języka polskiego.
