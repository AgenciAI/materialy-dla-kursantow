Instalacja środowiska animacji

HyperFrames · Remotion · video-use · FFmpeg · Claude Code

1
Homebrew
package manager dla macOS
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Jeśli już masz Homebrew, pomiń ten krok.

2
Node.js + npm
wymagany v18+
brew install node
3
FFmpeg
obróbka wideo
brew install ffmpeg
4
Claude Code CLI
główne narzędzie
npm install -g @anthropic-ai/claude-code
5

6
ElevenLabs API
wymagany dla video-use
  1. Zarejestruj konto na elevenlabs.io — darmowy tier wystarczy na start
  2. Wejdź w Settings → API Keys i skopiuj klucz
  3. Przy pierwszym użyciu /video-use Claude zapyta o klucz — wklej go wtedy
✓
Weryfikacja
sprawdź czy wszystko działa
node --version powinno być v18+
ffmpeg -version powinno pokazać wersję
npx hyperframes --version powinno pokazać wersję
